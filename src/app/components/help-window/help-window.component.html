1- UTILISATION

Recherche
Il est possible de faire une recherche par texte sur l'ensemble des données. La recherche n'est pas sensible aux majuscules/minuscules et s'applique à toutes les colonnes. Pour éviter certains faux positifs, il peut être nécessaire de désactiver certaines colonnes (surtout Utilisateur et Constitution). À noter que les champs Gagnant, Plateforme et URL sont ignorés.

Colonnes affichées
Dans le menu Champs, il est possible de activer/désactiver des colonnes. Lorsque désactivés, elles n'apparaisseront plus dans la table et la recherche ne prendra plus en compte les données qu'elles contiennent. Par défaut, toutes les colonnes sont affichées.

Ordre des colonnes
L'ordre des colonnes peut être modifié dans n'importe quel sens. Pour cela, maintenez le clic sur l'un des titres de colonne et faites le glisser le long de la ligne. À noter qu'il n'est pas possible de modifier l'ordre si une colonne est masquée.

Tri des colonnes
Les données d'une colonne peuvent être triés par ordre croissant ou décroissant en passant la souris sur le titre d'une colonne et cliquant sur l'icône de flèche apparaissant sur le côté. L'option de tri n'est pas disponible pour les champs Plateforme et URL.

Sauvegarde des paramètres
Certains paramètres sont sauvegardés dans le navigateur et s'applique d'une ouverte à l'autre de l'application. Pour le moment, cela englobe les paramètes suivants :
  • L'orde des colonnes.
  • Les colonnes affichés.
  • Le nombre de chanson affichés par page.

    3- LIMITATIONS

    L'application possède plusieurs limitations :
  • Les données ne sont pas mise à jours en temps réel. Il est donc possible d'avoir des décalages entre les constitutions ou avec le pseudo des utilisateurs.
  • Les données ne suivent pas toutes le même format. Il est possible de rencontrer des incohérences au niveau du champ Auteur qui peut regrouper un artiste ou un titre d'album. Il faut aussi prendre en compte le cas de plusieurs titres possible pour une même chanson (ex: Romanji vs Kanji).
  • Les données sont entrées par les utilisateurs du site Matbay et ne sont pas vérifiées, des erreurs peuvent s'y glisser.

    • 4- CONTRIBUTION

    Dans le cas d'une suggestion ou d'une erreur, vous pouvez contacter un admin Matbay ou ouvrir une issue sur le Github.
